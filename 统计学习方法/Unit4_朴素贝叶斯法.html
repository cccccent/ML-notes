第四章 朴素贝叶斯法

  • 朴素贝叶斯法是基于贝叶斯定理与特征条件独立假设的分类方法。
  • 朴素贝叶斯法与贝叶斯估计不同。

4.1  朴素贝叶斯法的学习和分类

4.1.1 基本方法

  • 输入：X⊆Rn\mathcal{X} \subseteq R^nX⊆Rn为n维向量集合
  • 输出：Y={c1,c2,...,ck}\mathcal{Y} = \{c_1,c_2,...,c_k\}Y={c1​,c2​,...,ck​}，输出为类标记，y∈Yy\in \mathcal{Y}y∈Y
  • 数据集：T={(x1,y1),(x2,y2),...,(xN,yN)}T = \{(x_1,y_1),(x_2,y_2),...,(x_N,y_N)\}T={(x1​,y1​),(x2​,y2​),...,(xN​,yN​)}由P(X,Y)P(X,Y)P(X,Y)独立同分布产生。
  • 训练过程：通过训练数据集，学习P(X,Y)P(X,Y)P(X,Y)（联合分布概率），也就是分别学习先验概率分布P(Y=ck),k=1,2,3...k先验概率分布P(Y = c_k),k = 1,2,3...k先验概率分布P(Y=ck​),k=1,2,3...k和条件概率分布P(X=x∣y=ck)=P(X(1)=x(1),...,X(n)=x(n)∣Y=ck}条件概率分布P(X =x|y = c_k) = P(X^{(1)}=x^{(1)},...,X^{(n)} = x^{(n)}|Y =c_k\}条件概率分布P(X=x∣y=ck​)=P(X(1)=x(1),...,X(n)=x(n)∣Y=ck​}
    • 但是这种条件概率的计算方式由指数数量的参数，其估计实际是不可行的。

朴素贝叶斯法：

  • 朴素贝叶斯法为了解决参数过多的问题，做出了条件独立性的假设：

P(X=x∣Y=ck)=P(x(1)=x(1),...,X(n)=x(n)∣Y=ck)=∏j=1nP(X(j)=x(j)∣Y=ck) P(X = x|Y = c_k) \\= P(x^{(1)}=x^{(1)},... ,X^{(n)}=x^{(n)}|Y = c_k) \\= \prod^n_{j=1}P(X^{(j)}=x^{(j)}|Y = c_k) P(X=x∣Y=ck​)=P(x(1)=x(1),...,X(n)=x(n)∣Y=ck​)=j=1∏n​P(X(j)=x(j)∣Y=ck​)

即假设用于分类的特征在类确定的条件下是相互独立的。

  • 后验概率的计算由贝叶斯定理得出：

P(Y=ck∣X=x)=P(X=x∣Y=ck)P(Y=ck)∑kP(X=x∣Y=ck)P(Y=ck)P(Y = c_k|X = x)=\frac{P(X = x|Y=c_k)P(Y=c_k)}{\sum_kP(X = x|Y=c_k)P(Y = c_k)} P(Y=ck​∣X=x)=∑k​P(X=x∣Y=ck​)P(Y=ck​)P(X=x∣Y=ck​)P(Y=ck​)​

代入条件独立性假设可得：

P(Y=ck∣X=x)=P(Y=ck)∏jP(X(j)=x(j)∣Y=ck)∑kP(Y=ck)∏jP(X(j)=x(j)∣Y=ck)k=1,2,...,KP(Y = c_k|X =x)=\frac{P(Y=c_k)\prod_jP(X^{(j)}=x^{(j)}|Y = c_k)}{\sum_kP(Y=c_k)\prod_jP(X^{(j)}=x^{(j)}|Y=c_k)}\\k = 1,2,...,K P(Y=ck​∣X=x)=∑k​P(Y=ck​)∏j​P(X(j)=x(j)∣Y=ck​)P(Y=ck​)∏j​P(X(j)=x(j)∣Y=ck​)​k=1,2,...,K

进而得到朴素贝叶斯分类器L：

y=f(x)=argmaxckP(Y=ck)∏jP(X(j)=x(j)∣Y=ck)∑kP(Y=ck)∏jP(X(j)=x(j)∣Y=ck)y = f(x)=argmax_{c_k}\frac{P(Y=c_k)\prod_jP(X^{(j)}=x^{(j)}|Y = c_k)}{\sum_kP(Y=c_k)\prod_jP(X^{(j)}=x^{(j)}|Y=c_k)} y=f(x)=argmaxck​​∑k​P(Y=ck​)∏j​P(X(j)=x(j)∣Y=ck​)P(Y=ck​)∏j​P(X(j)=x(j)∣Y=ck​)​

我们又知道，对于任意一个数据项或特征，上式分母都相同，所以：

y=argmaxckP(Y=ck)∏jP(X(j)=x(j)∣Y=ck)y= argmax_{c_k}P(Y = c_k)\prod_jP(X^{(j)}=x^{(j)}|Y = c_k) y=argmaxck​​P(Y=ck​)j∏​P(X(j)=x(j)∣Y=ck​)

4.1.2 后验概率最大化的含义

  • 这一部分主要介绍选择P(Y=ck∣X=x)P(Y = c_k|X = x)P(Y=ck​∣X=x)最大的ckc_kck​作为x的分类结果的合理性证明。
  • 假设选择0-1损失函数：

L(Y,f(x))={1,Y≠f(x)0,Y=f(x)L(Y,f(x))= \begin{cases} 1, Y\neq f(x) \\ 0, Y = f(x) \end{cases} L(Y,f(x))={1,Y​=f(x)0,Y=f(x)​

我们可以得到期望风险函数：

Rexp(f)=E[L(Y,f(X))]=EX∑k=1K[L(ck,f(X))]P(ck∣X)R_{exp}(f)=E[L(Y,f(X))]\\=E_X\sum^K_{k=1}[L(c_k,f(X))]P(c_k|X) Rexp​(f)=E[L(Y,f(X))]=EX​k=1∑K​[L(ck​,f(X))]P(ck​∣X)

为了使期望风险最小化，只需对X=xX = xX=x逐个极小化，得到：

f(x)=argmin⁡y∈Y∑k=1KL(ck,y)P(ck∣X=x)=argmin⁡y∈Y∑k=1KP(y≠ck∣X=x)=argmin⁡y∈Y(1−P(y=ck∣X=x))=argmax⁡y∈YP(y=ck∣X=x)f(x) = arg\min_{y\in \mathcal{Y}}\sum^K_{k=1}L(c_k,y)P(c_k|X = x) \\=arg\min_{y\in \mathcal{Y}}\sum^K_{k=1}P(y\neq c_k|X = x) \\=arg\min_{y\in \mathcal{Y}}(1-P(y=c_k|X = x)) \\=arg\max_{y\in \mathcal{Y}}P(y=c_k|X=x) f(x)=argy∈Ymin​k=1∑K​L(ck​,y)P(ck​∣X=x)=argy∈Ymin​k=1∑K​P(y​=ck​∣X=x)=argy∈Ymin​(1−P(y=ck​∣X=x))=argy∈Ymax​P(y=ck​∣X=x)

通过以上推导我们可以看出，当P(y=ck∣X=x)P(y=c_k|X =x)P(y=ck​∣X=x)取最大时，我们的期望风险将降到最低。这也就是朴素贝叶斯法所采用的原理。

4.2.3 贝叶斯估计

  • 在极大似然估计中，可能会出现所要估计的概率值为0的情况。这时会影响到后验概率的计算结果，使分类产生偏差。这一问题可通过贝叶斯估计解决。
  • 贝叶斯估计实际上是在每个频数上都加了一个相同的正数λ\lambdaλ，当λ=0\lambda = 0λ=0时，贝叶斯估计就是极大似然估计。
  • 通常我们将λ\lambdaλ取1，此式成为拉普拉斯平滑。并且可知，此式的贝叶斯估计是一种概率分布。
  • 条件概率的贝叶斯估计：

Pλ=(X(j)=ajl∣Y=ck)=∑i=1NI(xi(j)=ajl,yi=ck)+λ∑i=1NI(yi=ck)+Sjλ，λ≥0P_{\lambda}=(X^{(j)}=a_{jl}|Y=c_k)=\frac{\sum^N_{i=1}I(x_i^{(j)}= a_{jl},y_i=c_k)+\lambda}{\sum^N_{i=1}I(y_i=c_k)+S_j\lambda} ，\lambda≥0 Pλ​=(X(j)=ajl​∣Y=ck​)=∑i=1N​I(yi​=ck​)+Sj​λ∑i=1N​I(xi(j)​=ajl​,yi​=ck​)+λ​，λ≥0

  • 先验概率的贝叶斯估计：

Pλ(Y=ck)=∑i=1NI(yi=ck)+λN+KλP_\lambda(Y=c_k)=\frac{\sum^N_{i=1}I(y_i=c_k)+\lambda}{N+K\lambda} Pλ​(Y=ck​)=N+Kλ∑i=1N​I(yi​=ck​)+λ​
